# Brussels Airport Enterprise Network

Cisco Packet Tracer network design project — Odisee Campus Brussels (2024–2025)

## Overview
Enterprise network designed for a multi-site airport environment,
built as a group project during the Applied Informatics programme.

## Technologies
- Cisco routing & switching
- VLANs and inter-VLAN routing
- OSPF dynamic routing (multi-area)
- HSRP redundancy (high availability)
- ACLs and port security
- DHCP snooping
- SSH v2
- IPv4/IPv6 dual-stack
